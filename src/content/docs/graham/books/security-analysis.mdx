## 核心理念

### 1. 安全边际原则
安全边际是贯穿全书的核心思想：只有当价格显著低于内在价值时才买入。

### 2. 投资与投机的区分
- **投资**：通过深入分析，确保本金安全并获得合理回报
- **投机**：依赖短期价格波动，缺乏基本面支撑

### 3. 基本面分析优先
- 财务报表是企业价值的"体检报告"
- 关注资产、盈利、现金流等客观数据

### 4. 定量与定性结合
- 定量：财务数据、估值指标
- 定性：管理质量、行业前景
